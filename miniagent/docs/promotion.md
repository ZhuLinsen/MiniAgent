用[MiniAgent](https://github.com/ZhuLinsen/MiniAgent)打造AI助手超简单！三行代码就能搞定：
```python
from miniagent import MiniAgent

# 创建你的第一个Agent
agent = MiniAgent(
    model="deepseek-chat",
    api_key="your_api_key"
)

# 加载工具
agent.load_tools(["calculator", "web_search"])

# 运行Agent
response = agent.run("计算123 × 456并搜索最新的AI新闻")
print(response)
```
